track(stream, data)

Track data to Atom, this function returns a Promise which will be resolved only when data is tracked to backlog. The function rejects the promise in 3 cases: 1. Stream / Data are missing. 2. Tracker has been stopped. 3. Non-blocking mode and trackingTimeout has been reached.
Parameters:
Name Type Description
stream Atom stream name
data Data to track
